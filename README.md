## Goals for this repo
To create a series of Quantum Monte Carlo programs. The programs are for demonstration only.

## How to build:
```bash
make build
cd build
cmake .. -DCMAKE_TOOLCHAIN_FILE=/path/to/your/vckpg/vcpkg.cmake
```
